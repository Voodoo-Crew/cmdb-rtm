## :runner: Getting started ##

```shell
$ git clone https://github.com/tbaltrushaitis/cmdb-rtm
$ cd cmdb-rtm
$ cp -pr src/* ./
$ npm i
$ bower i
$ sudo chmod a+x app/bin/*.sh
$ npm start
```
